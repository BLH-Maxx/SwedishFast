Welcome to Swedish fast

The fastes way to learn swedish
The main idea behind swedish fast is that 20% of all swedish words represents 80% of the words we use day to day to communicate, and knowing that will give you above fluent capability of the language.

The word list is coming from thousands of wikipedia pages and are ranked using Zipfs Law

(DB yet to be currated)


Current Features

* Get 1 random word from the top 1000 most used words in swedish, get it translated to English and if so desired get an example of how to use it in a sentence(if we have it available)

* A RESTfull webservice API currently returning in JSON format
