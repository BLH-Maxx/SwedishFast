Wellcome to Swedish fast Api Docs



Currently we have 3 different



Getting a list of all basic swedish word
http://178.128.247.188:8001/basicwords

With this you will get the 1000 most used words in swedish language and we consider this to be the basics.

This api supports JSON



Getting a random word
http://178.128.247.188:8001/OneRandomWord

With this you will get the one random word out of the 1000 most used words in swedish language.

This api supports JSON



Getting a specific amount of random words
http://178.128.247.188:8001/SpecificRandomWords/{Ntimes}

With this you will get the specified amount of random word out of the 1000 most used words in swedish language.

This api supports JSON and dont forget to change {Ntimes} for a Integer between 1 and 1000
